🟤 Thin Mint Announcer's View
A compact, trackside CRG readout for announcers: clocks, score, lineups, jammer status, penalty heat, jam history, and the quick reference bits you need while the game is moving. Helps announcers to not just call the plays, but tell the story.
AAbout
Shows this help and version screen.
HJam History
Shows the full saved jam history for the game, separated by period.
PPenalty Codes
Shows the penalty code reference for the active CRG ruleset.
DDark / Light
Toggles the announcer view between dark mode and light mode.
